#API Endpoints

## HTML API

### ROOT
* `GET /`
  * loads the React web app

## JSON API

### USERS
* `GET /users`
  * Get all users
* `GET /users/:id`
  * Get user based on id
* `POST /users`
  * Create a new user
* `PATCH /users/:user_id`
  * Update a specific user

##  BOOK
* `GET /books`
  * `Get all books`
* `GET /book/search/:id`
  * `Get book based on title or author`
